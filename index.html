☁️ Google Drive 連携設定

商品・売上データを複数のiPadでリアルタイム共有できます。Google Cloud ConsoleでOAuth Client IDを取得してください（無料）。

1
Google Cloud Console でプロジェクトを作成し、Google Drive API を有効化
2
「APIとサービス」→「認証情報」→「OAuthクライアントID作成」。アプリの種類は ウェブアプリケーション を選択
3
「承認済みの JavaScript 生成元」にこのファイルを開いているURLを追加（例：https://xxx.github.io または file://）
4
発行されたClient IDを貼り付けて「保存して連携」を押す
正しいClient IDの形式ではありません（〜.apps.googleusercontent.com）
